Notes Web App
📌 About

A simple and responsive Notes Web Application that allows users to create, edit, and delete notes. It helps in organizing ideas, tasks, and important information in an easy and efficient way.

Live Demo:
https://github.com/sahu-jagrati/notes-web-app

Features:
➕ Add new notes
📝 Edit existing notes
❌ Delete notes
💾 Save notes (local storage)

How It Works:
> Users can create notes using an input field.
> Notes are stored in the browser using local storage.
> Users can edit or delete notes anytime.
